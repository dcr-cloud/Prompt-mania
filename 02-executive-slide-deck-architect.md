# 🏛️ Executive Slide Deck Architect

**Complexity:** ⭐⭐ Intermediate  
**Best For:** Converting reports, articles, or research into executive presentations  
**Source:** Report → Slides Conversion  
**Visual Style:** Professional & Analytical  
**Created by:** Dhruv Chauhan

---

## 🎯 Goal

Transform complex reports or articles into high-impact **10-slide executive decks** that communicate clearly and drive decisions.

---

## 🛠️ THE PROMPT

Copy everything below and paste into Claude / ChatGPT / NotebookLM:

```
# TASK
Convert this [REPORT/ARTICLE/VIDEO] into a 10-slide executive deck.

# SLIDE STRUCTURE (STRICT)
Each slide must follow this format:

HEADLINE: [6-9 words, action-oriented or insight-driven]
CORE MESSAGE: [One sentence, max 20 words]
SUPPORTING ELEMENTS: [Pick ONE]
→ 3-4 bullet points (each ≤10 words)
→ 1 data point + 1-line interpretation
→ 1 quote + context
→ 1 framework diagram (describe it)

VISUAL SUGGESTION: [Bar chart / Icon grid / Timeline / Comparison table]
```

---

## 🎨 Visual Philosophy

- **Analytical Tone:** Evidence-based and authoritative
- **Billboard Test:** Each slide must be comprehensible within 6 seconds
- **Sequential Flow:** Ideas must build logically from problem to action

---

## 📋 IDEAL SLIDE FLOW

```
SLIDE 1:  Executive Summary — The 1-sentence version of everything
SLIDE 2:  The Problem / Opportunity
SLIDE 3:  Current State (What's happening now)
SLIDE 4:  Root Cause Analysis
SLIDE 5:  Key Insight #1
SLIDE 6:  Key Insight #2
SLIDE 7:  Key Insight #3
SLIDE 8:  Recommended Solution
SLIDE 9:  Implementation Roadmap
SLIDE 10: Expected Outcomes / Next Steps
```

---

## ✅ Quality Checklist

Before finalizing, verify each slide:
- [ ] Headline is specific, not vague
- [ ] Content understood in 10 seconds
- [ ] Visual suggestion is actionable (not just "use a diagram")
- [ ] Takeaway can stand alone
- [ ] Advances narrative from previous slide

---

## 💡 How to Use

1. Paste this prompt into your AI tool
2. Add your report/article content after the prompt
3. Run it
4. Review each slide against the quality checklist
5. Ask the AI to revise any weak slides
