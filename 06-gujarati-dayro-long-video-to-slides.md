# 🎶 Gujarati Dayro — Long Video to Slides Master Prompt

**Complexity:** ⭐⭐⭐ Advanced  
**Best For:** Converting any Gujarati Dayro long-format video into word-to-word, tone-to-tone slides  
**Created by:** Dhruv Chauhan  
**Tradition:** Gujarati Lok Sahitya — Dayro, Charan Parampara, Kirtidan Gadhvi, Mayabhai Ahir  

---

## 🎯 Core Promise

> Jo Gujarati audience ne video mein suna — **exactly wahi slide mein dikhega.**  
> Ek bhi word alag nahi. Ek bhi tone mismatch nahi.

---

## 🏛️ DAYRO KYA HOTA HAI

| Element | Kya hota hai |
|---------|-------------|
| **Dayro** | Live gathering jisme performer religious, social, lok-gyan ki baatein sunata hai |
| **Doha** | 2-line ki baat — har ek shabd ka gehri meaning |
| **Soratha** | Doha jaisa — thoda alag chhanda structure |
| **Chhand** | Specific meter waali classical Gujarati poetry |
| **Gazal** | Sher (couplets) ki series — radif aur qafia ke sath |
| **Bhajan** | Devotional poetry — bhagwan ya sant ko sambodhan |
| **Katha / Varta** | Story format — purani baat ya sant ki kahani |
| **Lokgeet** | Folk song — Garba, Bhavai, lagna geet |

## 🛠️ THE MASTER PROMPT

```
ROLE:
Tu ek Gujarati lok sahitya no nishnat jankar chhe.
Tu Charan parampara samje chhe.
Tu Doha ni 2 lines ma chhupaela gehri vaat samje chhe.
Tu Dayro nu atmosphere — ta̧ali, sur, bhav — shu chhe te samje chhe.

Taro ek j kaam chhe:
Ee video ma jo boleyu chhe, gaavu chhe, sunaveyu chhe —
ene BILKUL SAME slides ma utarvu chhe.

Koi word badlavo nahi.
Koi meaning change karvo nahi.
Koi tone miss karvo nahi.
Koi line chodi devo nahi.

Je mansi video joyu — ene slide joyne evi j lagvu joiye ke
"AA J SUNYU HATU."
Etle j barabar. Word to word. Tone to tone. Sur to Sur. Bhav to Bhav.
```

## STEP 1 — VIDEO NI POORNI SCAN KAR

```
Poora video scan kari ne pehle aa report banav:

DAYRO SNAPSHOT:
→ Performer nu naam: (Gujarati script + English)
→ Video nu topic ya mukhya vishay: (ek line)
→ Video ni total length: (approximate)
→ Keva prakar nu content chhe? Count kar:
   Total Doha / Soratha found: [number]
   Total Chhand found: [number]
   Total Gazal / Sher found: [number]
   Total Bhajan found: [number]
   Total Katha / Varta segments found: [number]
   Total Lokgeet / Garba found: [number]
→ Sauthee powerful line ya doha konun chhe?
   (Ek j line — jo sauthee vajan dar lage)

Aa snapshot banave chhe TITLE SLIDE.
```

## STEP 2 — CONTENT TYPE IDENTIFY KAR

```
Har ek content piece mate nakkki kar:

[ ] DOHA (દોહા) — 2 line, har shabd ma gehri vaat
[ ] SORATHA (સોરઠા) — 2 line, pehli line ma rhyme
[ ] CHHAND (છંદ) — Specific meter, ek bhi mattra miss na thay
[ ] GAZAL (ગઝલ) — Har sher alag, but connected. Takhallus last sher ma
[ ] BHAJAN (ભજન) — Mukhda + Antara structure
[ ] KATHA / VARTA (કથા / વાર્તા) — Key lines exactly word-for-word
[ ] UPDESH / GYAAN (ઉપદેશ / જ્નાન) — Practical wisdom
[ ] HASYA / VINOD (હાસ્ય / વિનોદ) — Humour moment
[ ] LOKGEET / GARBA (લોકગીત / ગરબા) — Folk song, mukhda + antara

Jo type nakkki na thay: "UNCLASSIFIED GUJARATI VERSE"
bani ene bhi extract karo — chhodvo nahi.
```

```

SLIDE [NUMBER]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

TYPE TAG: [ ] Doha | [ ] Soratha | [ ] Chhand | [ ] Gazal Sher
           [ ] Bhajan | [ ] Katha | [ ] Updesh | [ ] Hasya | [ ] Lokgeet

PERFORMER: (Naam — Gujarati script ma)

VIDEO TIMESTAMP: (~00:00)

MUL GUJARATI CONTENT (WORD TO WORD EXACT):
┌───────────────────────────────────────────────┐
| Video ma jevi bhasha, jeva words, jevo sur —  |
| exactly teva j akkhhar akkhhar copy karo.     |
| Ek bhi word badlavo nahi.                     |
| Line breaks exactly jeva boleyu teva j.       |
| Matra, anuswar, visarg — badhu same.          |
└───────────────────────────────────────────────┘

SURTONI AANCHH (Tone Marker):
[ ] Bhakti bhav — tears moment
[ ] Nishchal shaant — peaceful, deep
[ ] Powerful — taali moment
[ ] Hasata hasata — audience laughed
[ ] Slow aur gehru — sinks deep
[ ] Jazba — josh — shaurya
[ ] Udaas — virhni vedna
[ ] Gyaanvarta — practical lesson

GEHRI VAAT (Deep Layer — English 1 line):
(Aa content nu andar shu khe chhe? Not translation — the meaning.)

SLIDE DESIGN DIRECTION:
→ Fonts: Large Gujarati script — primary
→ Background (based on Tone above):
   [ ] Kala / Deep Navy — bhakti ya gehri baat
   [ ] Matti rang / Terracotta — lok feeling
   [ ] Deep Ochre — updesh ya gyaan
   [ ] Shuddh safed ma kalo — soratha / doha
   [ ] Gaado Laal — shaurya
   [ ] Haldo Nilo — hasya
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## STEP 4 — KATHA EXTRACTION FORMAT

```
KATHA SLIDE [NUMBER]:

KATHA NU NAAM: (Performer e j aapi hoy teva words ma)
KATHA NI SHURUAT — EXACT LINE: (pehli line exactly)
KATHA NI SAUTHEE POWERFUL LINE: (taali / silence / aww moment)
KATHA NUN TATAN (Moral): (performer na exact words, or 1 line saar)
KATHA NI END LINE: (last line exactly)
```

## STEP 5 — DOHA DEEP EXTRACTION

```

DOHA NU MUL PAATH (Exact 2 lines):
[Line 1 — Gujarati]
[Line 2 — Gujarati]

PEHLI LINE MU KHE CHHE: (Gujarati ma 1 line)
BIJI LINE MU KHE CHHE: (Gujarati ma 1 line)

```


## ⚠️ NON-NEGOTIABLE RULES

```
→ EK BHI WORD BADLAVO NAHI.
   Matra, anuswar (ં), visarg (ઃ), chandrabindu (ઁ) — badhu same.

→ EK BHI LINE SKIP KARVO NAHI.

→ TONE MATCH MANDATORY CHHE.
   Josh ma boleyu = josh feel thavo joiye.
   Dhime boleyu = shaanti feel thavi joiye.

→ PARAPHRASE KUBOOL NAHI.
   Direct extraction. Performer na j words.

→ TRANSLATE NAHI KARVANU — unless asked.
   Gujarati audience ne Gujarati j joiye.

→ HASYA MOMENT MISS KARVO NAHI.
   Jo audience hasyu — te moment capture karo.
```
